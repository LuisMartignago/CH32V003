# 🚀 CH32V003 Projects

Repositório com exemplos, testes e projetos utilizando o microcontrolador **CH32V003 (RISC-V)**.

> Foco em código direto no hardware (baixo nível), sem dependência pesada de HAL.

---

## 📌 Sobre o CH32V003

O **CH32V003** é um microcontrolador RISC-V extremamente barato e poderoso:

- ⚡ Clock até 48 MHz  
- 💾 16 KB Flash / 2 KB RAM  
- 📡 Interfaces: USART, I2C, SPI  
- 🎛️ ADC 10 bits  
- ⏱️ Timers avançados (PWM, etc.)  
- 💲 Custo extremamente baixo  

Ideal para projetos embarcados simples, controle de potência, sensores e automação.
